## 🚀 Projets majeurs

### **Cibiyar Karatu — Plateforme d’apprentissage adaptatif (Capstone)**
- **Problème :** Les élèves ruraux manquent de parcours personnalisés ; les enseignants manquent d’outils de diagnostic ; connectivité limitée.  
- **Solution :** Microservices (API Gateway, Kafka, PostgreSQL + pgvector), moteur BKT/IRT, SRS, STT/TTS, sync offline mobile.  
- **Impact :** Rétention des acquis améliorée, enseignants plus efficaces, accessibilité accrue.  
- **Résultats :** Prototype fonctionnel ; tests simulant >10k sessions.  
- **Demo :** Screenshots/GIF sur [Portfolio](https://yacineseybousiddo.me/) — Full demo sur demande.

### **Commerce360 — Marketplace régional**
- **Problème :** Fragmentation des paiements, faible découverte cross-border, onboarding vendeur lent.  
- **Solution :** Microservices Node.js/React + Kafka, multi-paiement Stripe & iPay/CinetPay, AI TF-IDF + correction orthographique, ETL admin.  
- **Impact :** Onboarding vendeur 80 % plus rapide, commerce inter-pays facilité.  
- **Résultats :** Dev environment + Docker Compose, seed data 6 mois, dashboard analytics opérationnel.  
- **Demo :** Screenshots/GIF sur [Portfolio](https://yacineseybousiddo.me/) — Full demo sur demande.

### **OmniIntelOS — Plateforme d’intelligence d’entreprise**
- **Problème :** KPI fragmentés entre Finance, Opérations, RH ; besoin d’analyses prédictives localisées.  
- **Solution :** FastAPI + React, PostgreSQL + Chroma vector search, multi-source ingestion, forecasting Monte-Carlo, anomaly detection, 9-persona chatbot.  
- **Impact :** Décisions 10× plus rapides, coûts LLM réduits, conformité garantie.  
- **Résultats :** 25,920 KPIs testés, API p99 < 300ms, documentation et playbook inclus.
